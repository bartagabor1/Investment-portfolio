💼 Investment Portfolio Analysis

This project analyzes and visualizes data from an investment portfolio management system using MySQL, Python, Jupyter, and Power BI.

🧱 Project Structure

├── docker/                  #  MySQL init

🚀 Getting Started

1. Start the MySQL container

docker-compose up -d

Configuration in .env:

MYSQL_PORT=3307  
MYSQL_ROOT_PASSWORD=root  
MYSQL_DATABASE=investment_portfolio

2. Generate SQLite database

python create_db_from_sql.py

3. Generate Jupyter Notebook

python generate_notebook.py

Create notebook at: notebooks/analysis_investment_portfolio.ipynb

📊 Power BI Dashboards

🔑 KPI Highlights

  • Total AUM (Assets Under Management): Total value of all portfolios combined.
  • Average Annual Return: Mean annual return across all portfolios and investments.
  • Top Portfolio Return: Highest-performing portfolio based on annual return.
  • Risk-Adjusted Return: Return adjusted for volatility, providing a more balanced performance view.
